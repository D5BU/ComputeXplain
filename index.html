Welcome to ComputeXplain

Computers do millions of operations every split second that we take for granted. This project breaks down those invisible daily processes into high-fidelity, interactive, visual simulators. Explore network routing, CPU pipelines, subpixel display grids, and cache memory pipelines.

Network Packet Router

Watch how data is chopped into TCP packets, resolved via DNS layers, and routed dynamically across an active internet pathfinder graph.

CPU ALU & Pipeline

Execute instruction cycles manually or in auto mode. Watch bits travel through registers, program counters, and mathematical logic gates in real-time.

Pixel & Screen Raster

Draw on a canvas and magnify down to individual RGB subpixel LEDs. Test color-depth compression and watch progressive display sweep lines.

Cache Hierarchy

Simulate memory requests. Witness latency gaps as addresses search L1, L2, and L3 cache lines before resorting to RAM, maintaining hit/miss graphs.

Algorithm Visualizer

Trace sorting, searching, and pathfinding step-by-step. Visualize Bubble/Quick/Merge Sort, Dijkstra's algorithm, BFS, DFS, and Binary Search with active pseudocode tracing.
